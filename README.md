# 🗡️ 墨池剑冢 - 诗剑行

> 仗剑走天涯，诗酒趁年华

一个以武侠风格设计的学习激励网站，帮助你系统性地学习计算机知识，追踪学习进度，培养学习习惯。

## ✨ 核心特色

- 🎨 **新中式水墨风格** - 古风界面，沉浸式学习体验
- 🏔️ **境界修炼系统** - 9大境界等级，突破时有酷炫动画
- 🎁 **江湖奇遇机制** - 学习完成后随机触发奇遇，获得额外奖励
- 📖 **课程学习跟踪** - 系统管理课程和章节进度
- 📜 **每日修行清单** - 任务管理，支持优先级和难度设置
- 🏮 **连续打卡系统** - 培养学习习惯
- 📈 **数据可视化** - 热力图、趋势图、进度追踪

## 📚 功能模块

### 🏔️ 境界系统
| 境界 | 所需修为 | 称号 |
|------|---------|------|
| 🦐 江湖小虾 | 0 时辰 | 初入江湖 |
| 🐣 初窥门径 | 5 时辰 | 已入门墙 |
| 🐤 略有小成 | 20 时辰 | 稍有心得 |
| 🦅 融会贯通 | 50 时辰 | 触类旁通 |
| 🔥 炉火纯青 | 100 时辰 | 技艺精湛 |
| ⛰️ 登峰造极 | 200 时辰 | 造诣非凡 |
| 🧙 一代宗师 | 500 时辰 | 开宗立派 |
| 🌟 陆地神仙 | 1000 时辰 | 传说人物 |
| 🌙 天人合一 | 2000 时辰 | 超凡入圣 |

### 📖 三大武学秘籍（学习方向）
1. **👊 内功心法** - 计算机基础（操作系统、网络、数据结构）
2. **🗡️ 万剑归宗** - 大模型与多模态AI
3. **⚔️ 独孤九剑** - 强化学习

### 📚 课程学习跟踪
- 添加课程（如 CS144、CS61B）
- 为每门课程添加章节/Lecture
- 每个章节可附带笔记链接
- 实时显示完成进度
- 完成整门课程有庆祝动画

### 🏆 成就系统（15个）
- 打卡成就：初涉江湖、闻鸡起舞、寒窗苦读等
- 时长成就：面壁图破、磨穿铁砚、一代宗师等
- 专项成就：内功深厚、剑意通天、独孤求败等

### ⏱️ 闭关修炼（计时器）
- 预设时间：一炷香(25分)、一盏茶(45分)、一个时辰(60分)、九十息(90分)
- 快捷键支持：空格(开始/暂停)、R(重置)、F(专注模式)、Q(换名言)
- 呼吸灯效果、休息提醒

### 📊 数据可视化
- 热力图：过去91天的学习记录
- 趋势图：近7天/近30天学习统计
- 进度环：今日目标完成度

## ⌨️ 快捷键

| 按键 | 功能 |
|------|------|
| `空格` | 开始/暂停计时 |
| `R` | 重置计时器 |
| `F` | 进入/退出专注模式 |
| `Q` | 换一条名言 |
| `ESC` | 退出专注模式/关闭面板 |

## 💾 数据存储

所有数据保存在浏览器 LocalStorage：
- ✅ 完全离线可用
- ✅ 数据隐私安全
- ✅ 支持导出/导入备份
- ⚠️ 清除浏览器数据会丢失记录

## 🌐 部署方式

### GitHub Pages（推荐，免费）
1. 在 GitHub 创建仓库
2. 上传所有文件
3. Settings → Pages → 选择 main 分支
4. 获得 `https://你的用户名.github.io/仓库名` 链接

### Vercel（免费，自动部署）
1. 注册 Vercel 账号
2. 导入 GitHub 仓库
3. 自动部署，获得 `.vercel.app` 域名

### Netlify（免费，拖拽部署）
1. 访问 netlify.com
2. 直接拖拽项目文件夹上传
3. 获得 `.netlify.app` 域名
